This plugins pack adds next assets to your Photoshop app:
- VTF texture format support;
- DDS texture format support;
- Height map to Normal map converter;
- Texture tiling plugin (Tiler).

Installation:
------------
Extract contents of "photoshop.7z" archive into your Photoshop app location,
for example, "C:\Program Files\Adobe\Photoshop CS6\", replace exising files.

When the SDK is installing, it finds app's path and installs it automatically.
Do the manual extraction only if it wasn't happen correctly!
