This plugins pack adds next assets to your Paint.NET app:
- VTF texture format support;
- Height map to Normal map converter;
- Alpha-Displacement Mask effect.

Installation:
------------
Extract contents of "paintdotnet.7z" archive into your Paint.NET app location,
for example, "C:\Program Files\PaintDotNet\", replace exising files.

When the SDK is installing, it finds app's path and installs it automatically.
Do the manual extraction only if it wasn't happen correctly!
